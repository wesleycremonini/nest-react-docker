# Sobre
Esse projeto utiliza Docker para rodar a aplicação localmente em 3 containers.

<h3>backend api (NestJS)</h3>
<h3>frontend web (ReactJS)</h3>
<h3>database (MySQL)</h3>

# Requisitos
Docker e docker compose.

# Como rodar
1. Clonar o repositório.
2. Tenha certeza de que as portas locais 80 e 3001 estão vagas
3. 
```bash
docker compose up --build
```
4. Pronto, é só acessar o localhost
